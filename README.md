## maude-mode
This is a WIP maude-mode for my emacs config. Note that, although basic funcionality is implemented, it currently works for my own purposes while I write my master's thesis, and I'll work on new features as I need them.

This is intended to work in hand with [maude-ts-mode](https://github.com/luisvgs/maude-ts-mode), which I also maintain.

### Feature roadmap

* [x] open-repl.
* [x] load-file.
* [ ] rewriting interaction within the REPL.
* [ ] kill REPL instanced properly.
* [ ] inline sort/type annotations.
